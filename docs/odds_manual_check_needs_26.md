# needs_manual_check 26レースのオッズ手動確認

作成日: 2026-06-18

## 目的

`data/reports/missing_win_odds_races_classified.csv` で `reason_candidate = needs_manual_check` だった26レースについて、公式ページ上の `win_odds_final` 欠損理由を確認した記録です。

この確認では、既存CSVの修正、`win_odds_final` の補完、予想モデル作成、特徴量追加、自動取得機能の作成は行いません。

## 確認方法

確認対象:

- 成績ページ: `RaceMarkTable`
- 参考確認: `OddsTanFuku`

確認した観点:

- 公式成績ページに `単勝オッズ` 列が存在するか
- 欠損対象馬の成績ページ上の単勝オッズ値が空欄か
- `OddsTanFuku` に該当馬の単勝オッズ値が存在するか
- CSV上の欠損が `input_missing` と見なせるか
- 成績ページとは別の正規ソース参照が必要と見なせるか

## 結論

26レースはすべて `page_has_no_odds` と分類します。

確認した26レースでは、成績ページにも `OddsTanFuku` にも欠損対象馬の行は存在しましたが、該当馬の単勝オッズ値は空欄でした。公式ページ上に値が存在するのにCSVだけ空欄、というケースは確認されなかったため、`input_missing` はありません。

また、`OddsTanFuku` 側でも該当馬の単勝オッズ値は空欄だったため、今回の26件については `different_source_needed` には分類しません。

## 次の注意点

- 今回の26件は、通常出走馬のオッズ欠損ではなくSCR/EXC馬のオッズ欠損です。
- `input_missing` は確認されていないため、既存CSVの `win_odds_final` は修正しません。
- 2025-10〜2026-01の214レースは、代表10件から `page_has_no_odds` の可能性が高いものの、全件確認済みではないため引き続き未確定として扱います。
